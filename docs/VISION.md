# MANARAH

## Vision

MANARAH is an Educational Knowledge Operating System.

Its mission is to transform educational content into structured knowledge that can be understood, connected, searched, explained, and taught by AI.

## Core Principle

Everything is Knowledge.

Books are only one source of knowledge.

## Knowledge Hierarchy

Book
 └── Lesson
      ├── Concept
      ├── Rule
      ├── Definition
      ├── Example
      ├── Exercise
      ├── Question
      └── Relation
